Detailed Description

Class description and usage The fault handler should be implemeted in every class which has "something which might go wrong" This could be: 1) semaphores -> thread-related issues
2) sensors -> bad measurements / hanging
3) timeouts -> stuck in a endless loop
4) other ....

@Class constructor: Class A () : m_faultHandler{"Class A"}

@Faulty code: void faulty_function if(bug){ m_faultHandler.handleFault( { FaultHandler_n::severity::very_high,"faulty_function","we have found a bug" } ); }
